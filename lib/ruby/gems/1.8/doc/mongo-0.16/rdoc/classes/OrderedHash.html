Class OrderedHash
In: lib/mongo/util/ordered_hash.rb
Parent: Hash

A hash in which the order of keys are preserved.

Under Ruby 1.9 and greater, this class has no added methods because Ruby‘s Hash already keeps its keys ordered by order of insertion.

    def self.[] *args
      oh = OrderedHash.new
      if Hash === args[0]
        oh.merge! args[0]
      elsif (args.size % 2) != 0
        raise ArgumentError, "odd number of elements for Hash"
      else
        0.step(args.size - 1, 2) do |key|
          value = key + 1
          oh[args[key]] = args[value]
        end
      end
      oh
    end

    def merge!(other)
      @ordered_keys ||= []
      @ordered_keys += other.keys # unordered if not an OrderedHash
      @ordered_keys.uniq!
      super(other)
    end
