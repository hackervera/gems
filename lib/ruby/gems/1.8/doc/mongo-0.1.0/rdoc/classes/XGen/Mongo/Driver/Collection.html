clear()

Remove all records.

count(selector={})

Return the number of records that match selector. If selector is nil or an empty hash, returns the count of all records.

create_index(name, *fields)

Create a new index named index_name. fields should be an array of field names.

drop_index(name)

Drop index name.

drop_indexes()

Drop all indexes.

find(selector={}, options={})

Return records that match a selector hash. See Mongo docs for details.

Options:

:fields :Array of collection field names; only those will be returned (plus _id if defined)
:offset :Start at this record when returning records
:limit :Maximum number of records to return
:sort :Either hash of field names as keys and 1/-1 as values; 1 == ascending, -1 == descending, or array of field names (all assumed to be sorted in ascending order).
hint(hint_fields)

Set hint fields to use and return self. hint_fields may be a single field name, array of field names, or a hash whose keys will become the hint field names. May be nil.

index_information()

Return an array of hashes, one for each index. Each hash contains:

:name :Index name
:keys :Hash whose keys are the names of the fields that make up the key and values are integers.
:ns :Namespace; same as this collection‘s name.
insert(*objects)

Insert objects, which are hashes. "<<" is aliased to this method.

modify(selector, modifier_obj)

Update records that match selector by applying obj as an update. Both selector and modifier_obj are required.

options()

Return a hash containing options that apply to this collection. ‘create’ will be the collection name. For the other possible keys and values, see DB#create_collection.

remove(selector={})

Remove the records that match selector.

replace(selector, obj)

Update records that match selector by applying obj as an update.

repsert(selector, obj)

Update records that match selector by applying obj as an update. If no match, inserts (???).
